# Tyrian
[![Build Status](https://travis-ci.org/rubenvanassche/Tyrian.svg)](https://travis-ci.org/rubenvanassche/Tyrian)
A game for advanced programming

See Doc/Report.pdf for design choices

## Dependencies
- C++11 Compiler
- SFML 2.1

## Build
```
cmake .
make
```

## Running
The game:
```
./Tyrian
```
The tests:
```
cd Library
./LibTests
```
